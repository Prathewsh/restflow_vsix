# RestFlow

RestFlow is a lightweight REST client integrated directly into VS Code as a webview panel. It allows you to test APIs, send requests, and view responses without leaving your editor.

## Features

- **Request Builder**: Select HTTP methods (GET, POST, PUT, DELETE, etc.) and enter URLs.
- **Body Support**: Send Raw JSON or Form Data.
- **Response Viewer**: View status codes and formatted JSON/Text responses.
- **Integrated UI**: Styled to match VS Code's dark theme.

## Usage

1. Open the Command Palette (`Ctrl+Shift+P`).
2. Run the command **Open RestFlow**.
3. Enter your API URL and click **Send**.

## Extension Settings

This extension currently has no configurable settings.
